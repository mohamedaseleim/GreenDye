## Color Scheme

```
Components use Material-UI theme:
- Primary Color: #2e7d32 (Green)
- Secondary Color: #ff9800 (Orange)
- Background: #ffffff (White)
- Cards: Slight shadow with rounded corners
- Dragging: Elevated shadow, slightly transparent background
- Success indicators: Green chips
- Warning indicators: Orange/yellow background
- Error indicators: Red text/icons
```

## Accessibility Features

- **Keyboard Navigation:** Tab through all interactive elements
- **Screen Reader Support:** ARIA labels on all buttons and form fields
- **Visual Indicators:** Clear focus states on all interactive elements
- **Color Contrast:** Meets WCAG AA standards
- **Drag Handle:** Clear visual indicator for draggable items

## Animation States

1. **Drag Start:** Item elevates with shadow
2. **Dragging:** Item follows cursor with reduced opacity
3. **Drop Zone:** Shows insertion point with visual indicator
4. **Drop Success:** Item settles into new position
5. **Save:** Brief loading indicator, then success message

---

This UI provides an intuitive, accessible, and visually appealing interface for managing course sections and lessons.
